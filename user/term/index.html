이용방법

음원을 실시간으로 다운받거나 스트리밍이 가능하고, 영상을 보고 퀴즈를 풀 수 있습니다.



개발 관련 안내

이 웹사이트는 Bootstrap을 사용한 Mobirise Engine으로 디자인되었으며,

웹사이트의 기능들은 방송부 1~3기 부장 이정윤이 개발하였습니다.

Github Pages을 통해 웹사이트는 24시간 호스팅되고 있습니다.
